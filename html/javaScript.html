Funciones de Orden Superior

Funciones que reciben o retornan otras funciones:

console.log(triplicar(5));  // 15

// Funci√≥n que recibe otra funci√≥n (callback)

    const resultado = [];
    for (let elemento of array) {
        resultado.push(operacion(elemento));

üí° Consejo: Usa arrow functions para funciones cortas y callbacks. Usa funciones tradicionales cuando necesites this o arguments.
